# Nexor Roadmap

## Phase 1 — Operational Stability

Core features required for production use.

* standardized meter calculation
* standardized export formats
* directory structure normalization
* centralized formatting functions

---

## Phase 2 — Production Traceability

Advanced tracking capabilities.

* machine identification via ComputerName
* audit-safe correction system
* immutable production records
* event-based change tracking

---

## Phase 3 — Operational Intelligence

Transform production data into insights.

* productivity per machine
* time per meter metrics
* bottleneck detection
* production efficiency metrics

---

## Phase 4 — Production Planning

Production queue planning system.

* import print images
* calculate length using DPI
* group jobs by fabric
* manage roll capacity
* insert automatic gaps
* estimate total production time

---

## Phase 5 — SaaS Architecture

Evolution to cloud architecture with local agent.

Nexor Agent responsibilities:

* monitor machine log folders
* automatically import logs
* normalize printer formats
* operate offline when required
* synchronize data when connection returns

---

## Long Term Vision

Evolve Nexor into a platform capable of:

* production optimization
* intelligent queue suggestions
* waste analysis
* predictive production planning
